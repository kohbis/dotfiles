## Usage

### Setting

```bash
$ git clone https://github.com/kohbis/dotfiles.git your-workspace/dotfiles
$ cd your-workspace/dotfiles
$ ./setup.sh [option]
```

| Option | Description |
|--------|-------------|
| `-c` | Create symbolic links |
| `-d` | Delete symbolic links |
| `-r` | Recreate symbolic links (delete & create) |
